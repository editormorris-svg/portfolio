# VP Prompt Library — Local AI Usage

## Incident Report Drafting

```
# PROMPT: Incident Report Drafting
# NEVER include student names — use "Student A" or case numbers

Draft a professional incident report with these details:
- Date: [INSERT]
- Location: [INSERT]
- Involved parties: Student A (Grade 7), Student B (Grade 7)
- Witnesses: Teacher A, Teacher B
- Incident type: [physical/verbal/other]
- Brief description: [2-3 sentences, factual]

Include: summary, action taken, follow-up required, parent notification status.
```

## Parent Communication

```
# PROMPT: Parent Communication
# Remove all identifying information before processing

Draft a professional email to parents regarding:
- Topic: [academic concern / behavioral update / positive news]
- Tone: [formal / supportive / collaborative]
- Key points: [bullet points, anonymized]
- Requested action: [if any]

Length: 150-200 words. Include opening, details, next steps, closing.
```

## Staff Meeting Agenda

```
# PROMPT: Staff Meeting Agenda

Create a structured staff meeting agenda with:
- Meeting duration: [60/90 minutes]
- Topics: [list of general topics, no names]
- Priority: [high/medium/low for each]

Format: Title, time allocation, discussion points, action items template.
```
