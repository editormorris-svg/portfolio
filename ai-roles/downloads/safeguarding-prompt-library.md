# Safeguarding Prompt Library — Local AI Usage

## CP Report Summary

```
# PROMPT: CP Report Summary
# NEVER include names, dates, or identifying details

Summarize this child protection incident for:
- Case number: [ANONYMIZED REFERENCE]
- Incident type: [general category only]
- Actions taken: [general description]
- Follow-up required: [yes/no + general type]

Remove: names, dates, specific locations, medical details.
```
