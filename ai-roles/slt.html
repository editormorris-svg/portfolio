⚠️ The Danger: How SLT Currently Uses Cloud AI

Real Incident (Names Changed)

A Principal used ChatGPT to draft a strategic plan for next academic year. The document included:

  • Staff restructuring plans with names and positions
  • Budget allocations and cost-cutting measures
  • Student demographic data and enrollment projections
  • Competitive analysis of other schools

Result: Sensitive strategic data processed on US servers. Potential data breach. No control over retention.

Common Cloud AI Misuse by SLT

TaskWhat Gets LeakedRisk Level
Strategic planning documentsSchool vulnerabilities, competitive intelligenceCRITICAL

✅ The Solution: Local AI for SLT

Why Local AI Changes Everything

  • Zero Data Exposure: Strategic plans never leave the server room
  • Full Audit Trail: Every query logged with your user ID and timestamp
  • Role-Based Access: Only SLT can access strategic data
  • PIPL Compliant: Data never leaves China's jurisdiction
  • Competitive Advantage: Plans cannot be scraped by competitors

📋 Detailed Usage Scenarios

STRATEGIC

Strategic Plan Drafting

Current (Dangerous): Upload full plan to cloud AI for "review"

Local AI Method: Use anonymized prompts with specific sections

# PROMPT: Strategic Plan Section # Remove school name, staff names, and financial specifics Review this strategic plan section for: - Clarity and specificity - Measurable outcomes - Realistic timelines - Missing elements Section: [PASTE ANONYMIZED SECTION] Suggest: improvements, additional considerations, potential risks.
📥 Download Template
POLICY

Policy Development

Current (Dangerous): Use cloud AI to draft entire policies

Local AI Method: Generate framework, then add specific details manually

# PROMPT: Policy Framework # Generate generic framework, then customize locally Generate a policy framework for: - Topic: [data protection / safeguarding / acceptable use] - Audience: [staff / students / parents] - Length: [brief / comprehensive] Include: purpose, scope, responsibilities, procedures, review cycle.
📥 Download Template
BOARD

Board Report Preparation

Current (Risky): Use cloud AI to summarize board reports

Local AI Method: Summarize anonymized data points

# PROMPT: Board Report Summary # Use only anonymized data points Summarize this board report for: - Key achievements - Challenges faced - Financial overview (anonymized figures) - Next steps Data points: [ANONYMIZED BULLET POINTS] Format: Executive summary, 2-3 paragraphs.
📥 Download Template
INSPECTION

Inspection Preparation

Current (Risky): Upload inspection documents to cloud AI

Local AI Method: Generate self-assessment from anonymized criteria

# PROMPT: Inspection Self-Assessment # Based on general criteria, not specific school data Generate a self-assessment framework for: - Inspection area: [leadership / teaching / safeguarding] - Key criteria: [list from framework] Include: evidence required, potential gaps, improvement actions.
