⚠️ Known Issues

Red API Error

Status: Non-critical

Symptom: thinking is enabled but reasoning_content is missing

Impact: HER still functions. Can be fixed by setting reasoning: false in openclaw.json if it becomes annoying.

Git Push Interruptions

Status: Intermittent

Symptom: SIGTERM during push after commit succeeds.

Workaround: Run git push origin main separately after commit.
