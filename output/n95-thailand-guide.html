📋 What is N95?

N95 is a portable mini PC that will serve as your media and streaming companion in Thailand.

  • Hardware: Intel N95 CPU, 32GB RAM, 477GB SSD

  • Secondary: Photo/video backup, file sharing, remote access

🎬 Disney Plus Streaming

Option 1: Browser (Recommended)

  1. Open Google Chrome on N95
  2. Navigate to disneyplus.com
  3. Log in with your Disney Plus account
  4. Start streaming!

Option 2: VLC Media Player

  1. Open VLC (pre-installed)
  2. Media → Open Network Stream
  3. Enter URL if available
⚠️ VPN Note: Disney Plus may require a US/UK IP address. If content is blocked:
  • Use Tailscale with a US exit node (if configured)
  • Or use a travel VPN service
  • Hotel WiFi in Thailand may work without VPN

🔧 Troubleshooting

Can't connect to Disney Plus?

  • Check internet: ping 8.8.8.8
  • Try Chrome (not Firefox)
  • Check if VPN needed for content
  • Try mobile hotspot vs hotel WiFi

Can't reach N95 from home/school?

  • Check N95 is online: tailscale status on N95
  • Check N95 has internet: ping 8.8.8.8
  • Check firewall: sudo ufw status
  • Restart Tailscale: sudo systemctl restart tailscaled

Disk full?

  • Check: df -h
  • Clean: rm -rf /media/thailand/movies/old-file.mp4
  • Offload to HIM: rsync -avz /media/thailand/photos/ dell01@100.71.125.55:/mnt/data8tb/n95-backup/
