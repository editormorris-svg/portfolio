How to Use

Step 1: Create Dell01 Image

⏳ Pending
  1. Plug USB #1 into Dell01
  2. Power on Dell01, press F12
  3. Select USB UEFI boot
  4. Clonezilla boots automatically
  5. Select: device-image → local_dev → /dev/sdb2
  6. Choose folder: /images
  7. Mode: saveparts → name: dell01-golden
  8. Select source: /dev/sda2
  9. Wait 15-30 minutes

Step 2: Copy to USB #2 & #3

⏳ Pending
  1. Boot Dell01 normally (remove USB #1)
  2. Plug USB #1, #2, #3 into Dell01
  3. Mount all three: /dev/sdb2, /dev/sdc2, /dev/sdd2
  4. Copy image: sudo cp -r /media/usb1/images/dell01-golden /media/usb2/images/
  5. Copy image: sudo cp -r /media/usb1/images/dell01-golden /media/usb3/images/
  6. Wait ~10 minutes per copy

Step 3: Parallel Clone

⏳ Pending
  1. USB #1 → Dell02
  2. USB #2 → Dell03
  3. USB #3 → Dell04
  4. Power all on, press F12 on each
  5. Select USB UEFI boot on each
  6. Clonezilla boots → device-image → local_dev
  7. Mode: restoreparts → select dell01-golden
  8. Target: internal disk (/dev/sda or /dev/nvme0n1)
  9. Wait 15-20 minutes per laptop
