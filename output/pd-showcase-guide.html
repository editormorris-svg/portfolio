11. 安全与防火墙

组件状态功能
ufw（防火墙）Active阻止所有入站，除 SSH、Tailscale、Samba 外
SSH 密钥认证Enabled无密码登录 — 仅加密密钥
unattended-upgradesEnabled自动安装安全补丁
pam_sss.soDisabled修复阻止登录的 PAM 错误配置
NOPASSWD sudoConfigureddell01 无需密码提示即可运行管理员命令
教师培训安全要点：
  • "每台机器默认都有防火墙"
  • "SSH 无需密码 — 仅密钥，如同数字指纹"
  • "更新自动发生 — 无需 IT 人员"

13. 教师培训演示场景

六个脚本化演示。每个 3–5 分钟。适合教师培训或管理展示。

📄

场景 A：替代 Microsoft Office

5 分钟
  1. 打开 LibreOffice Writer
  2. 创建教案
  3. 保存为 .docx
  4. 在 Windows 机器上打开
  5. "免费。兼容。无订阅。"
🤖

场景 B：本地机器上的 AI

5 分钟
  1. 打开 Jupyter Notebook
  2. 运行 Python 词频脚本
  3. 展示词汇使用饼图
  4. "无需云端，无需 API 密钥，无需互联网"
🎥

场景 C：录制课程

5 分钟
  1. 打开 OBS Studio
  2. 点击"开始录制"
  3. 讲授 30 秒微型课
  4. 点击"停止录制"
  5. 在 mpv 中播放视频
🀄

场景 D：随处输入中文

3 分钟
  1. 打开 LibreOffice Writer
  2. 按 Ctrl+Space
  3. 输入拼音：nihao
  4. 从候选词选择 你好
⌨️

场景 E：终端高级用户

5 分钟
  1. neofetch → 显示规格
  2. trans :zh "Welcome to class" → 即时翻译
  3. espeak-ng "Welcome to class" → 文本转语音
  4. docker run hello-world → 5 秒容器
🖥️

场景 F：远程管理

5 分钟
  1. 从 HER（另一台机器）SSH 进入 Dell01
  2. 如同坐在键盘前运行命令
  3. "我可以在客厅修复所有 13 台机器"

⚡ 基本命令

系统信息

neofetch
htop

开发

jupyter notebook
# 盒子中的网页服务器
docker run -p 8080:80 nginx
# 终端中的 JavaScript

# 显示 Tailscale IP
tailscale ip -4

iperf3 -c <server-ip>
