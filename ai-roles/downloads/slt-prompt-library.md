# SLT Prompt Library — Local AI Usage

## Strategic Plan Section

```
# PROMPT: Strategic Plan Section
# Remove school name, staff names, and financial specifics

Review this strategic plan section for:
- Clarity and specificity
- Measurable outcomes
- Realistic timelines
- Missing elements

Section: [PASTE ANONYMIZED SECTION]

Suggest: improvements, additional considerations, potential risks.
```

## Policy Framework

```
# PROMPT: Policy Framework
# Generate generic framework, then customize locally

Generate a policy framework for:
- Topic: [data protection / safeguarding / acceptable use]
- Audience: [staff / students / parents]
- Length: [brief / comprehensive]

Include: purpose, scope, responsibilities, procedures, review cycle.
```

## Board Report Summary

```
# PROMPT: Board Report Summary
# Use only anonymized data points

Summarize this board report for:
- Key achievements
- Challenges faced
- Financial overview (anonymized figures)
- Next steps

Data points: [ANONYMIZED BULLET POINTS]

Format: Executive summary, 2-3 paragraphs.
```

## Inspection Self-Assessment

```
# PROMPT: Inspection Self-Assessment
# Based on general criteria, not specific school data

Generate a self-assessment framework for:
- Inspection area: [leadership / teaching / safeguarding]
- Key criteria: [list from framework]

Include: evidence required, potential gaps, improvement actions.
```
