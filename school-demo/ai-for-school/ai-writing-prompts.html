Why Use AI Writing Prompts?

AI can be your teaching assistant—if you know how to ask. The right prompt transforms vague AI responses into classroom-ready materials. Below are battle-tested templates organized by teaching task.

1. Lesson Plan Generation

Standard Lesson Plan

🎯 Copy This Prompt

Create a 45-minute lesson plan for [GRADE LEVEL] on [TOPIC/TEXT]. 

Requirements:
- Learning objective aligned with [SPECIFIC STANDARD, e.g., CCSS.ELA-LITERACY.RL.9-10.2]
- Opening hook (5 min) that connects to students' lives
- Main activity (25 min) with step-by-step instructions
- Formative assessment (10 min)
- Exit ticket question
- Differentiation for EAL learners and advanced students

Context: My class has [NUMBER] students, mixed ability, [SPECIFIC CONTEXT].

💡 Example Output Preview

Lesson: Grade 10 - Theme in "Of Mice and Men"

Hook: "Think of a time you dreamed of something that didn't happen. Share with a partner."

Main Activity: Gallery walk with 6 quote stations, students identify theme evidence...

Literature Circle Lesson

🎯 Copy This Prompt

Design a 5-day literature circle unit for [BOOK TITLE] at [GRADE LEVEL].

Each day should include:
- Reading assignment (page/chapter breakdown)
- Role sheet (Discussion Director, Vocabulary Enricher, Illustrator, Connector, Summarizer)
- Discussion questions at 3 complexity levels
- Mini-lesson on a literary device

Final product: Group presentation option.

2. Rubric Creation

Analytical Rubric

🎯 Copy This Prompt

Create an analytical rubric for [ASSIGNMENT TYPE] on a 4-point scale (Exemplary, Proficient, Developing, Beginning).

Categories:
- Content/Understanding
- Organization/Structure
- Use of Evidence
- Language/Conventions

For each category, include:
- Specific descriptors at each level
- Point values
- "Look-fors" for quick assessment

Topic: [SPECIFIC ASSIGNMENT DETAILS]

Single-Point Rubric (For Feedback Focus)

🎯 Copy This Prompt

Create a single-point rubric for [ASSIGNMENT].

Format:
- Middle column: Proficient criteria (what success looks like)
- Left column: Concerns area (what's missing)
- Right column: Advanced area (exceeds expectations)

Include 4-5 criteria and space for teacher comments.
💡 Pro Tip: Single-point rubrics reduce grading time by 40% while providing clearer feedback. Students focus on one target instead of deciphering multiple levels.

3. Student Feedback & Comments

Report Card Comments

🎯 Copy This Prompt

Write 3 variations of report card comments for a student who [PERFORMANCE DESCRIPTION].

Tone: Professional, constructive, specific
Length: 2-3 sentences each
Include: Strength, area for growth, next step

Example performance: Strong reader, struggles with written expression, participates well in discussions

Assignment Feedback

🎯 Copy This Prompt

Provide feedback on this student writing sample using the "Glow, Grow, Goal" format:

[GIVE AI THE STUDENT WORK]

Glow: What they did well (specific, with evidence)
Grow: One specific area to improve
Goal: Actionable next step

Keep tone encouraging. Use second person ("You...")

Participation Feedback

🎯 Copy This Prompt

Generate 10 comment stems for different participation patterns:

1. Strong contributor who dominates
2. Quiet but thoughtful when called on
3. Engaged listener, rarely volunteers
4. Distracted/disengaged
5. Collaborative group member

Format: "[Student] consistently..." or "I've noticed [Student]..."
Include specific behavior examples in each.

4. Differentiated Assignments

Tiered Assignment

🎯 Copy This Prompt

Create 3 versions of an assignment on [TOPIC]:

Level 1 (Approaching): Scaffolding provided, sentence starters, graphic organizers
Level 2 (At Grade): Standard expectations
Level 3 (Advanced): Open-ended, extension questions, creative options

Same learning objective, different complexity. Include:
- Clear instructions for each level
- Time estimates
- Success criteria

Choice Board

🎯 Copy This Prompt

Design a 3x3 choice board for [UNIT/TEXT] with 9 different activities.

Format options:
- Visual (draw, design, map)
- Written (essay, letter, diary entry)
- Kinesthetic (drama, debate, hands-on)
- Tech (video, podcast, slideshow)

Each activity should:
- Connect to same learning objectives
- Take approximately same time
- Have clear success criteria

6. AI Prompt Engineering Tips

🔑 The RICE Framework for Better Results:

Role - Tell the AI who it is ("You are an experienced EAL teacher...")
Instructions - Be specific about what you want
Context - Give background on your students, school, situation
Examples - Show samples of what good looks like

Before vs After

❌ Weak Prompt

Write a lesson plan about Romeo and Juliet.

✅ Strong Prompt

You are an experienced high school English teacher with 15 years in EAL classrooms.

Create a 50-minute lesson on dramatic irony in Romeo and Juliet Act 3, Scene 1 for Grade 10 mixed-ability learners.

Requirements:
- Hook using a modern TV/movie clip comparison
- Students identify 3 examples of dramatic irony in the scene
- Partner activity with sentence frames for EAL support
- Exit ticket: "Explain how dramatic irony creates tension for the audience"

My class: 24 students, 8 EAL learners (intermediate), 4 reluctant readers.

7. Subject-Specific Prompts

Science

🎯 Copy This Prompt

Create a lab report template for [GRADE] with:
- Section-by-section guidance
- Sentence starters for hypothesis, procedure, conclusion
- Checklist for self-assessment
- Exemplar paragraph for each section

Make it accessible for EAL students without oversimplifying the science.

History/Social Studies

🎯 Copy This Prompt

Generate a Document-Based Question (DBQ) for [HISTORICAL EVENT]:
- 2-3 primary source documents (include excerpts)
- Scaffolding questions for each document
- Essay prompt with clear thesis requirement
- Graphic organizer for evidence gathering

Math

🎯 Copy This Prompt

Create error analysis problems for [MATH TOPIC]:
- 3 worked examples with common mistakes
- Students identify the error, explain why it's wrong, solve correctly
- Reflection question: "What mistake are you most likely to make? How will you check?"

8. Saving & Organizing Your Prompts

Build your personal prompt library:

  1. Create a Prompt Bank: Use Notion, Google Docs, or even a simple spreadsheet
  2. Tag by Use: Lesson plans, assessment, differentiation, parent comms
  3. Include Context: Note what worked, what to tweak
  4. Version Control: Mark prompts as v1, v2 as you refine them
  5. Share with Colleagues: Build department-level prompt libraries

Start Small, Build Momentum

Pick ONE prompt from this page. Use it this week. Tweak it. Next week, add another. In a month, you'll have a custom toolkit that saves hours every week.
